# Yutnori

## Description

	(yutnori)[https://en.wikipedia.org/wiki/Yunnori] is a traditional korean board game usually played during the new year.

## Features

- Game Server (Authoritative Model). 
- Desktop Game Client.
- Room System.

## Usage

- to run the server
```bash
  make run_server
```

- to run a client
```bash
  make run_client
```
